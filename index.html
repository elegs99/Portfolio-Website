EasyMeshVR

1 / 3
Move vertices, edges, and faces by simply reaching out and grabbing them
2 / 3
Hop into a multiplayer lobby to collaborate on your edits using in-game voice chat
3 / 3

❮ ❯

EasyMeshVR is an open-source VR program built in Unity to target the Steam and Meta platforms. From ideation to execution, this project was completed entirely by me and my team over 6 months. This virtual environment takes advantage of the best features of VR to make editing 3D models easy such as the use of intuitive hand gestures to quickly move, rotate, and scale the model. Other key features include collaborative editing in real-time using multiplayer lobbies and quickly transferring 3D model files using our web server.


The freedom of the VR platform combined with the ability to define our requirements makes this the ideal project for a creative problem solver like myself. Due to my previous experience working on Nutrition Intuition, I was voted by my group to be the project manager. My main goals as project manager were to set the requirements for EasyMeshVR and track each task needed to build the final product.

❮ ❯

In the beginning of 2018, I became interested in game programming and started making small projects in Unity. My first exploration of machine learning, more specifically reinforcement learning, occurred during this time with Evolutionary Rockets. Evolutionary Rockets is a 2D simulation where 40 small rockets have to find their way to a target using limited knowledge of their environment. Over many generations, these rockets learn to avoid obstacles and refine their path to take as few frames as possible.


If you want to play around with the demo I updated the UI and upload a webGL build on this website. You can click the controller icon below to access it.

Nutrition Intuition is a full-stack web development project that helps users track the nutritional value of the food they eat. The project was completed over 10 weeks through collaboration with 5 other students. We had a lot of freedom in deciding how to set up our web server and ultimately decided to develop Nutrition Intuition on the MERN stack due to its ease of use and reliability.


As the project manager, I adhered to agile development principles. Requirements were gathered early and thoroughly, we met frequently with all stakeholders, and avoided any periods of crunch. This project challenged me in scale and complexity, but it also made me much more comfortable with front-end development and database management. In the end, we had a fully functional platform with a web and mobile interface.
